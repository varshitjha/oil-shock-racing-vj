# ⚡ Oil Shock Racing

> Oil price shock simulation platform by ZeTheta Algorithms.

## Quick Start
```bash
npm install
npm start
```

## Structure
```
src/
  constants/    ← static data
  engine/       ← simulation logic
  hooks/        ← React state wrappers
  styles/       ← theme + CSS
  components/   ← UI (ui/, charts/, layout/, tabs/)
  App.jsx       ← root
  index.js      ← entry
```
